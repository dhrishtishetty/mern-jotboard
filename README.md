# 📝 MERN JotBoard

A full-stack note-taking application built using the MERN stack. JotBoard allows users to create, edit, and delete notes through a clean and responsive interface.

## 🚀 Features

* Create new notes
* Update existing notes
* Delete notes
* Responsive user interface
* Real-time interaction with a MongoDB database
* RESTful API architecture

## 🛠️ Tech Stack

### Frontend

* React
* React Router
* Axios
* Tailwind CSS
* DaisyUI

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/dhrishtishetty/mern-jotboard.git
cd mern-jotboard
```

### Install dependencies

```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

### Run the application

```bash
# Backend
npm run dev

# Frontend
npm run dev
```
